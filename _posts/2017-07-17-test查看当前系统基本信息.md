查看cpu
```bash
cat /proc/cpuinfo
```

查看内存
```bash
cat /proc/meminfo
```

查看服务器型号
```bash
dmidecode | grep 'Product Name'    
```

查看系统版本
```bash
cat /proc/version
```

查看机器名、并且可以修改
```bash
cat /etc/hostname
```

系统os错误代码查询
```bash
perror 24
OS error code  24:  Too many open files
```
